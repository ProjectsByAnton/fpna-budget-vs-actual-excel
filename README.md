This project demonstrates a **Financial Planning & Analysis (FP&A)** workflow using Microsoft Excel to evaluate **budget vs actual performance**, identify **cost variance drivers**, and generate **management-ready insights**.

---

## Project Overview
- Built an end-to-end **Budget vs Actual analysis** using Excel
- Analysed spending performance across **categories and time**
- Identified key **cost overruns and underutilisation areas**
- Designed **pivot tables, variance metrics, and charts** for decision-making

---

## Key Analysis & Techniques
- Budget vs Actual comparison
- Variance & Variance % calculation
- Budget utilisation analysis
- Pivot tables by month and category
- Visual dashboards (trend & variance charts)

---

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Financial variance analysis techniques (FP&A)

---

## Data Source
- Public dataset sourced from **Kaggle**
- Synthetic financial data used for educational and analytical purposes

> Note: The dataset is simulated and used solely to demonstrate FP&A concepts.

---

## Output
- Excel file: `FPnA_Budget_vs_Actual_Analysis.xlsx`
- Includes structured data, calculations, pivot tables, charts, and summary insights
